CROSS Wave 신규 서비스 출시

프로젝트 기간

  • 2025.09.02 ~ 2025.09.30

역할

  • Product Manager

  • 상위 정책기획, 프론트엔드, 백오피스 기획

배경

  • 글로벌 블록체인 MMORPG 로한2 게임 론칭을 앞두고 게임 인지도 제고 및 커뮤니티 활성화 필요

  • 기존 게임 크리에이터 생태계(유튜브, 트위치, 킥) 중심에서 자사 플랫폼으로 유입 필요성 대두

  • 촉박한 일정(4주)으로 인한 최소 기능 제품으로 구현

기획 프로세스

  • 타 게임사 크리에이터 플랫폼 5종 벤치마킹

    • 넥슨 크리에이터즈, 크래프톤 KCN, 에픽게임즈 SAC, 호요크리에이터즈, 넷마블 파트너 크리에이터즈 플러스

  • MoSCoW 방식으로 핵심 기능 도출 :

    • 상금누적(수익화)
    • 크리에이터 후원(플랫폼 기능)
    • 크리에이터 영상 업로드(콘텐츠 플랫폼화)
    • 크리에이터 랭킹(경쟁심리 자극)

  • 4주 일정을 감안하여 정산, 자동 크리에이터 등록 등의 기능은 Phase 2로 후순위 설정

실행 및 협업

  • 사업팀과의 5회 협의를 통한 서비스 정책 수립

    • 상금 누적 구조(인게임 상품 결제금액의 5%)
    • 랭킹 정책
    • 크리에이터 플랫폼(유튜브, 트위치, 틱톡, 페이스북, 바이낸스 스퀘어, 킥)
    • 최소 후원 수량

  • Figma로 서비스 전체 페이지 기획 및 디자이너와 즉각적인 논의를 통한 빠른 피드백 반영

  • 개발 기간 단축을 위한 병렬식 프로세스 채택 및 우선순위 조정

  • QA 기간 단축을 위해 QA 프로세스에 투입하여 30건의 JIRA 티켓 생성

성과

  • 크리에이터 확보 : 목표 300명 대비 293% 달성(880명)

  • 후원 생태계 활성화 :

    • 평균 후원율 66%
    • 일 평균 후원 수량 : 1,458 DROP
    • 1회 이상 후원한 팬 비율 52%
    • 4주 촉박한 일정 내 성공적인 서비스 출시

기술 역량

  • 기획 : Figma

  • 협업 & 문서화 : JIRA, Confluence, Slack

  • 데이터 분석 : Python(Pandas, Numpy)
